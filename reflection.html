Responsive Design

Definition

Responsive design is a term used in web design entailing on how to create web pages that look good on all devices, regardless of the screen size. Responsive web design automatically adjuts for different screens without compromising the quality of the design. The main concept here is using HTML and CSS to automatically resize, hide, shrink, and/or enlarge a web page: in the end making it look and feel good on all devices ranging from phones, tablets, laptops, desktops and other display projections. With the current technology, most people have access to smart phones and it should be possible to visit websites and enjoy the view from the phone screens. This is what mostly necessitates the need for responsive web designs.

  • Why Responsive Design in Web Technologies?

    In web design, endless innovations of different resolutions and devices keep emerging on a daily basis. Creating a new website version whenever an innovation is made on resolutions or different devices would almost an impossibility to achieve. To address these challenges, there should be a way in which a website should automatically respond to the needs of the user and the devices they are using. By this we mean that the website layout changes based on the size and capabilities of the screen that the users have.

    Different devices have different screen sizes. For instance, we have phones, phablets, tablets, desktops, game consoles, TVs, watches, LCD(Liquid Crystal Display), TFTs(Thin Film Transistors) among others. These screen sizes and resolutions are always in a state of change and thus any good website should adapt automatically to fit the new screen sizes.
